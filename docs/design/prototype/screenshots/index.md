# screenshots 索引

本层保存 `docs/design/prototype/` 下 standalone HTML 原型的浏览器渲染截图，用于评审和对齐页面布局。

## 截图来源

- 正式截图直接打开 standalone HTML，不从 `overview.html` iframe 截取。
- Desktop 标准 viewport：`1440x1000`。
- Mobile 标准 viewport：`390x844`。
- `overview.html` 只作为总览评审入口，用于按页面分组比较 desktop/mobile 结构关系。
