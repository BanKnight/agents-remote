# docs 索引

本层用于组织长期沉淀内容。

## 子目录

- [templates](./templates/) — 长期文档模板目录。
- [specs](./specs/) — 长期 WHAT：能力规格、行为契约、可验证需求。
- [design](./design/) — 长期 design：从运行态 design 提炼出的设计内容。
- [architecture](./architecture/) — 长期 HOW：系统级架构、模块边界、集成模式与 ADR。
- [runbooks](./runbooks/) — 运维、故障、迁移、发布等操作手册。

## 文档

- [AGENTS.md](./AGENTS.md) — 定义 `docs/` 的长期文档治理规则。
- [project.md](./project.md) — 项目认知 big picture，记录项目定位、用户场景、领域概念与长期准则。
